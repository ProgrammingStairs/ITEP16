Control Statement : The statement or instructions are used to inform the computer to perform the task when a program is being executed. There are four typees of control statement : a. Sequential control statement input ----- process(formula implementation) ----- output b. Decision | Conditional control statement i. if statement ii. if else statement iii. nested if else statement iv. else if ladder c. Looping control statement i. for loop ii. while Loop iii. do while loop iv. for each loop v. for in loop (indexing) vi. for of loop (value) d. Switch control statement
